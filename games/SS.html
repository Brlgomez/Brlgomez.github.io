January - March 2015

Javascript, HTML, CSS

By Brandon Gomez, Eron Lake, Antony Robbins, and Darion Toney

An isometric, real-time, Oregon Trail-like game set in the post apocalyptic future where water is a scarcity.

Role: Our team decided not to use an engine so I made the engine from scratch. In addition, I programmed the transformation of the world in order to give the game an isometric view and I programmed the resource management so the game can be efficient. I also made the art direction and created some sprites. I also implmented the draw battle system, the simple ai. and the statistical elements of the game like water and health.
