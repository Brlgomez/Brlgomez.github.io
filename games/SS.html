Scorching Sea (PC, March 2015)

Language: Javascript, CSS

Team Size: 4

Roles: Game designer, programmer, and artist

An isometric, real-time, Oregon Trail-like game set in the post apocalyptic future where water is a scarcity. Since our team did not use any engines, I made the engine from scratch. I programmed the transformation of the world in order to give the game an isometric view and I programmed the collision detection, and resource management so the game can be efficient. I also made the art direction and some sprites.
